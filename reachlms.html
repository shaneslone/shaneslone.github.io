Reach LMS

Reach LMS is an open source learning platform targeted at the developing world. The goal is for any organization to be able to use Reach LMS to create their own online learning platform. Reach LMS was designed using React.js, Redux, and Ant Design on the frontend, along with a Java Spring backend using a PostgresSQL database.
